# Fuentes de datos

## ERA5
- Reanálisis climático del ECMWF
- Variables: temperatura, humedad, presión, etc.

## AEMET
- Estaciones meteorológicas en España
- API opendata

## Open-Meteo
- API meteorológica gratuita
- Usada para forecasts

## OpenUV
- Radiación UV
- API con límite gratuito

## MoMo (ISCIII)
- Mortalidad diaria por todas las causas
- Usado como target para riesgo
